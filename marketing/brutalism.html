Why teams switch

Monitoring that never leaves the building.

Most monitoring tools ship your traffic metadata to someone else's cloud. BlueEye doesn't. It's a single on-prem server plus lightweight agents — you keep the data, the keys, and the audit trail.

🏢

On-prem by design

One Node.js + MySQL server on your own hardware. No SaaS account, no outbound telemetry, no third-party data processor to vet.

🕵️

Privacy first

Metadata only — ports, ASN, timings, the 5-tuple. Never deep packet inspection. Private/RFC1918 addresses are never geolocated.

🧮

Explainable analysis

Robust statistics (median + MAD z-score), not an opaque ML model. Every finding carries a plain-language reason and its evidence.

🇪🇺

EU sovereignty

European or self-hosted map tiles, GeoIP, fonts. No US-based vendors in the stack. Built for GDPR and data-residency requirements.

🛰️

Air-gap ready

Licenses verify offline with an embedded Ed25519 public key. Signed offline license files run in fully disconnected sites.

📊

NIS2 built in

Risk register, control evidence, incident deadlines and signed management reports — compliance isn't a bolt-on, it's a tab.

What's in the box

One platform. Every layer of visibility.

Lightweight agents report traffic, flows, system health and active probes to the server over an authenticated WebSocket. The dashboard turns it into answers.

🌍

Geo & ASN map

See internal hosts and external destinations on a European map. Drill into a country or ASN to the exact flows and findings behind it.

🚨

Alerting

Route findings to email, webhook or syslog with per-channel rules. Test any channel with one click before you rely on it.

🤖

Opt-in AI assistant

An optional, license-gated assistant explains a finding in context — on your terms, disabled by default, fully auditable.

🗺️

Topology & paths

Path visualization and topology views show how traffic actually moves between locations, agents and destinations.

🔐

RBAC + audit

Three roles (viewer / operator / admin), JWT sessions, bcrypt hashes, and a change trail on every compliance action.

📁

Export anything

CSV or JSON export on findings, geo, agents, locations and traffic. Print-ready HTML reports become PDFs in one step.

Live in an afternoon

From zero to visibility in four steps.

01

Install the server

Node.js + MySQL on your own box. Run the migrations, start the service. Docker Compose if you prefer.

02

Drop in your key

Set LICENSE_KEY — the only value a customer must configure. The server binds and verifies offline.

03

Enroll agents

Generate one-time codes, run the installer on each host. Agents self-enroll and start reporting over a secure channel.

04

Watch & comply

Findings, maps and NIS2 evidence fill in automatically. Tune alerts, approve reports, export the audit trail.
