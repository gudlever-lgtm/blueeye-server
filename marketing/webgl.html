Why teams switch

Monitoring that never leaves the building.

Most tools ship your traffic metadata to someone else's cloud. BlueEye doesn't. One on-prem server plus lightweight agents — you keep the data, the keys, and the audit trail.

🏢

On-prem by design

One Node.js + MySQL server on your own hardware. No SaaS account, no outbound telemetry, no third-party processor to vet.

🕵️

Privacy first

Metadata only — ports, ASN, timings, the 5-tuple. Never DPI. Private / RFC1918 addresses are never geolocated.

🧮

Explainable analysis

Robust statistics (median + MAD z-score), not an opaque model. Every finding carries a reason and its evidence.

🇪🇺

EU sovereignty

European or self-hosted tiles, GeoIP and fonts. No US-based vendors in the stack. Built for GDPR and data residency.

🛰️

Air-gap ready

Licenses verify offline with an embedded Ed25519 key. Signed offline license files run in fully disconnected sites.

📊

NIS2 built in

Risk register, control evidence, incident deadlines and signed reports — compliance is a tab, not a bolt-on.

The platform

One platform. Every layer of visibility.

Lightweight agents report traffic, flows, system health and active probes over an authenticated WebSocket. The dashboard turns it into answers.

🌍

Geo & ASN map

Internal hosts and external destinations on a European map. Drill into a country or ASN down to the exact flows and findings.

🚨

Alerting

Route findings to email, webhook or syslog with per-channel rules. Test any channel with one click before you rely on it.

🤖

Opt-in AI assistant

An optional, license-gated assistant explains findings in context — disabled by default, fully auditable, on your terms.

🗺️

Topology & paths

Path visualization shows how traffic actually moves between locations, agents and destinations.

🔐

RBAC + audit

Three roles, JWT sessions, bcrypt hashes, and a change trail on every compliance action.

📁

Export anything

CSV or JSON on findings, geo, agents, locations and traffic. Print-ready HTML reports become PDFs in one step.

Live in an afternoon

From zero to visibility in four steps.

STEP 01

Install the server

Node.js + MySQL on your own box. Run migrations, start the service. Docker Compose if you prefer.

STEP 02

Drop in your key

Set LICENSE_KEY — the only value you must configure. The server binds and verifies offline.

STEP 03

Enroll agents

Generate one-time codes, run the installer. Agents self-enroll and start reporting over a secure channel.

STEP 04

Watch & comply

Findings, maps and NIS2 evidence fill in automatically. Tune alerts, approve reports, export the audit trail.
